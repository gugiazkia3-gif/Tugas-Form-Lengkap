## Tema dan Desain
Aplikasi ini menggunakan palet warna premium untuk kesan yang profesional:
* Deep Blue (#1A237E): Mewakili stabilitas dan profesionalisme.
* Soft Sand / Tan (#D7CCC8): Memberikan kontras yang hangat dan elegan.
* Tipografi: Font bersih dengan komponen Material Outlined untuk keterbacaan tinggi.

## Fitur Utama
Proyek ini mengimplementasikan seluruh persyaratan sesuai panduan tugas:

1. Formulir Lengkap (01): Bidang registrasi terintegrasi menggunakan TextInputLayout dan TextInputEditText untuk Nama, Email, dan Password.
2. Validasi Lanjutan (02):
    * Pengecekan format email secara real-time.
    * Deteksi input kosong.
    * Logika validasi minimal pilihan pada Checkbox.
3. Kontrol Seleksi (03): Implementasi RadioGroup untuk jenis kelamin dan Checkbox untuk hobi (minimal 3 pilihan).
4. Spinner dan Dialog (04):
    * Custom adapter untuk pemilihan data Kota.
    * AlertDialog konfirmasi saat tombol submit diklik.
5. Interaksi Gesture (05): Penambahan gesture Long Press (tekan lama) pada tombol Submit untuk memicu aksi reset formulir.
6. Layout Responsif: Dibungkus dalam ScrollView untuk memastikan kompatibilitas di berbagai ukuran layar.

## Tech Stack
* Bahasa: Kotlin
* UI Framework: Android XML (Material Components)
* IDE: Android Studio
* Arsitektur: View Binding / Manual Binding
